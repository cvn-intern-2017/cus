1. Introduction

This is URL Shorterner app, which is the Intern Project at Cybozu Vietnam, on July, 2017. This app can create short link, which help user to not remember long links, complex links. Our product will help people to improve their user experience.

2. User 's Guide

You can type or paste the url you want to shorten this form. We only handle valid tasks, so you should enter the correct url in the format. Some of the rules when entering url is as follows:

  • The input character can not contain the whole number.
  • Contain Hyper Text Transfer Protoco.
  • Not contain space.
  • String length that user input into less than 256 characters.

Click the button "Shorten URl" to process the request and we will return the shortcut that you need below.

The red button when clicked will access the analysis page for the link you created

We have given you statistics on the number of times the url has been visited, the browser used to access it
